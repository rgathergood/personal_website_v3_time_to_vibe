# Rob Gathergood - Personal Portfolio

A modern Next.js portfolio website showcasing product management expertise, music, and renewable energy projects.

## 🚀 Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Hosting:** Vercel (recommended)

## 📋 Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Dark mode aesthetic
- ✅ Project showcase with detailed pages
- ✅ Blog functionality
- ✅ Contact form
- ✅ SEO optimized
- ✅ TypeScript for type safety
- ✅ Fast performance with Next.js

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/rob-portfolio.git
cd rob-portfolio

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000 in your browser
